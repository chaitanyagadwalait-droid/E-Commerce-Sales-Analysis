# 📌 Project Overview

This project combines **Data Analytics** and **Machine Learning** to understand customer purchasing behavior and support data-driven business decisions.

The workflow begins with raw transactional data, followed by SQL-based data preparation, feature engineering, exploratory data analysis, predictive modeling, and interactive Power BI dashboard development.

The project demonstrates a complete analytics lifecycle from data collection to business insights.

---

# 🎯 Business Problem

E-commerce companies need to answer questions such as:

* Which customers generate the highest revenue?
* What factors influence customer spending?
* Which customers are likely to become high-value customers?
* How can customer segmentation improve marketing campaigns?
* Which purchasing behaviors impact business growth?
* How can predictive analytics improve customer retention?
